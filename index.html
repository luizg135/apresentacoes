O Prompt Inicial (O Pedido)

Para um bom resultado, seja claro com a IA. Especifique o formato, o design e o conteúdo.

ChatGPT / Gemini
Aja como um desenvolvedor web sênior especialista em UI/UX. Crie uma apresentação de slides em HTML, CSS e JavaScript. Regras absolutas e inegociáveis: 1. Todo o código (HTML, estilo e scripts) deve estar em um ÚNICO arquivo chamado index.html. 2. Use as cores institucionais do Sistema FAEP (Verde #00703c, tons de cinza e branco). 3. Adicione botões interativos para avançar e voltar os slides. 4. Crie 3 slides explicativos sobre "Sustentabilidade no Agronegócio". 5. O design deve ser moderno, usar a fonte 'Inter' e ser totalmente responsivo.

A Regra de Ouro

Sempre reforce a exigência do "ÚNICO arquivo index.html". Isso elimina a complexidade de gerenciar pastas de imagens, arquivos CSS e JS separados na hora da publicação web.

Erro: "Page Not Found"

O erro clássico nº 1. A página simplesmente não abre ao clicar no link.

Diagnosticando o Erro
Fiz o upload no Netlify, mas recebo o erro "Page Not Found". O único arquivo que enviei chama-se "Apresentacao_Final.html".

Resposta:

Servidores web procuram por padrão um arquivo chamado index.html (tudo minúsculo).
Solução: Renomeie "Apresentacao_Final.html" para index.html e envie novamente.

Erro: "Imagens Quebradas"

A apresentação abre, mas as fotos viram ícones quebrados ou não carregam na internet.

Diagnosticando o Erro
Minha apresentação está online no Netlify, mas a foto que eu coloquei aparece quebrada. O caminho no código está: src="C:/Users/Documentos/foto.png".

Resposta:

O Netlify não tem acesso ao disco `C:/` do seu computador. Para imagens funcionarem na web, use links online (URLs como https://site.com/foto.jpg).

Se quiser usar um arquivo local, salve a imagem na mesma pasta do index.html, altere o código para src="foto.png" e faça o upload da pasta inteira no Netlify.

Erro: "Tudo quebrado no Celular"

A apresentação ficou linda no computador, mas ao enviar o link pelo celular, os textos vazam da tela.

Como pedir correção
O código atual ficou ótimo no computador, mas no celular o texto ultrapassa a tela e cria uma barra de rolagem horizontal. Corrija o CSS para ser 100% responsivo.

Resposta (Ação da IA):

(A IA irá reescrever seu código aplicando regras de media queries CSS).

Dica Pro: Sempre inclua a palavra "Responsivo" ou "Mobile-first" no seu Primeiro Prompt para evitar retrabalho.
